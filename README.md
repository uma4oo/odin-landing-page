# Odin Landing Page

A simple responsive landing page built for the Odin Project. The page presents a yoga studio experience with a hero section, feature cards, inspirational quote, and call-to-action.

Repository: https://github.com/uma4oo/odin-landing-page.git

Live demo: https://uma4oo.github.io/odin-landing-page/

## Project structure

- `index.html` — main HTML markup for the landing page
- `style.css` — page styling and layout
- `logo_yoga.png` — header logo graphic
- `wellness.jpg` — hero image
- `Goals.jpg`, `Grace.jpg`, `Growth.jpg`, `Gratitude.jpg` — feature card images
- `01.png`, `02.png` — additional image assets included in the repository

## Features

- Header with logo and navigation links
- Hero section with welcome text, description, and signup button
- Feature cards highlighting goals, grace, growth, and gratitude
- Inspirational quote section
- Bottom call-to-action area with signup button
- Simple, clean styling for a yoga studio landing page

## How to view

1. Open `index.html` in a web browser.
2. Or use a local development server to preview the site.

## Notes

- This project is a static HTML/CSS landing page.
- No JavaScript is required.
